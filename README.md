# 2D Platform Game - Sleutels & Diamanten

Een leuke 2D platformgame gemaakt met Python en Pygame!

## Features

✨ **Gameplay:**
- Verzamel sleutels (goud) om deuren te openen
- Verzamel diamanten (cyaan) om in de shop te kopen
- Beweeg je naar de gouden exit-doos om naar het volgende level te gaan
- 3 uitdagende levels met toenemende moeilijkheid

🎮 **Controls:**
- **A/D of Pijltjes Links/Rechts** - Bewegen
- **SPATIE/W/Pijltje Omhoog** - Springen
- **E** - Open shop
- **ESC** - Sluit shop / Afsluiten

🛍️ **Shop (Druk E):**
1. **Speed Boost** - 50 diamanten (beweeg sneller)
2. **High Jump** - 75 diamanten (spring hoger)
3. **Shield** - 100 diamanten (bescherming)

## Hoe te spelen

1. Installeer Pygame:
```bash
pip install pygame
```

2. Start het spel:
```bash
python main.py
```

## Level Progressie

- **Level 1**: Maak jezelf vertrouwd met de besturing
- **Level 2**: Meerder sleutels en deuren nodig
- **Level 3**: Eindchallenge met veel platforms en collectibles

## Tips

💡 Sparen voor upgrades in de shop maakt moeilijkere levels makkelijker!
💡 Je kunt sleutels meerdere keren oppikken.
💡 Val niet in het gat!
